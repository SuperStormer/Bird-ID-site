Usage

This bot uses bird images and sounds from Cornell's Maucalay Library to help you practice identifying them!

Discord Commands:

  • Type b!b or b.b for a new bird.
  • Type b!c (answer) or b.c (answer) to check your answer.
  • Type b!s or b.s for a bird call.
  • Type b!cs (answer) or b.cs (answer) to check the song.
